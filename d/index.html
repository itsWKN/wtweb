WinTroy Documentation

Command
copy

Description: Copies a file.

Example: copy "c:\sus.txt" "c:\users\batman\desktop\homework.txt" false

Arguments:

  • source-File-Path (text): The source file path.
  • dest-File-Path (text): The destination file path.
  • overwrite (boolean): Whether to overwrite if the destination file already exists.
get-file

Description: Retrieves a file.

Example: get-file "c:\users\batman\desktop\homework.txt"

Arguments: file-Path (text): The file path to retrieve.
move

Description: Moves a file.

Example: move "c:\users\batman\hehe.exe" "c:\users\batman\app.exe"

Arguments:

  • source-File-Path (text): The source file path.
  • dest-File-Path (text): The destination file path.
remove

Description: Removes a file.

Example: remove "c:\users\batman\desktop\homework.txt"

Arguments: file-Path (text): The file path to remove.
end

Description: Ends the process.

Example: end

Arguments: None
restart

Description: Restarts the process.

Example: restart

Arguments: None

Description: Opens a file.

Example: open "explorer" "c:\windows" false

Arguments:

  • file-Path (text): The file path to open.
  • args (text, null): Additional arguments for opening the file.
  • hidden (boolean, false): Whether to open the file hidden.

cd

Description: Change the current directory the program is in.

Examples:
cd "c:\windows"
cd ..
cd homework
cd

Arguments: cd (text): Directional Command:
jump into a subdirectory by typing a directory name,
jump into a directory by typing a directory path,
type .. to jump into the next lower layer subdirectory,
let the argument blank to retrieve the current directory
